# IGLU Movies

## BUILT WITH
* ReactJS
* Vite

## INSTALLATION
To clone into your local computer then run in terminal:
```sh
  git clone https://github.com/stenolmre/iglu-movies.git
```

In order to use movies endpoint, please add .env to root and inside the file, please add api key:
```env
  VITE_API_KEY=
```

To run it locally:
```sh
  npm run install
```
```sh
  npm run dev
```
